Multi-Agent Travel Planning Assistant

An intelligent multi-agent travel planning system built with LangGraph that coordinates specialized AI agents to create personalized travel plans. The assistant searches for flights and hotels, generates optimized itineraries, and delivers a unified travel experience while maintaining conversation memory.

✨ Features
✈️ Flight Search Agent – Finds available flight options based on user preferences.
🏨 Hotel Recommendation Agent – Suggests hotels according to budget, location, and ratings.
🗺️ Itinerary Planning Agent – Creates personalized day-wise travel itineraries.
🤖 Travel Coordinator Agent – Combines outputs from all agents into a comprehensive travel plan.
🧠 Persistent Conversation Memory using PostgreSQL.
🌐 Real-Time API Integration for travel information.
💻 Interactive Streamlit Dashboard for a user-friendly experience.
